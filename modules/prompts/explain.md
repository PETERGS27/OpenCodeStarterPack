# Explain Prompt

Explain code or concepts in detail.

## For code

```
## Purpose
What does this code do? (1-2 sentences)

## How it works
Step-by-step walkthrough of the logic.

## Key details
- Line X-Y: what this section does
- Function Z: what it returns, side effects
- Data structure W: why this was chosen

## Context
- Why is this approach used?
- What alternatives exist?
- What are the limitations?

## Visual
[If helpful, include a diagram or flow]
```

## For concepts

```
## Definition
What is it? (1-2 sentences)

## Core idea
The fundamental concept explained simply.

## Deeper
- How it works under the hood
- Key principles and mechanisms

## Why it matters
- What problem does it solve?
- When would you use it?

## Related
- Related concepts
- Prerequisites to understand this
- Where to go next
```
